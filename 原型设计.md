# 原型设计

## 1. 页面结构

```
┌─────────────────────────────────────────────────────┐
│                     顶部导航栏                        │
│  Logo  |  首页  |  设备管理  |  报警中心  |  工单  |  我的  │
└─────────────────────────────────────────────────────┘
```

## 2. 页面清单

| 序号 | 页面名称 | 路径 | 说明 |
|------|----------|------|------|
| 1 | 登录页 | /login | 用户登录 |
| 2 | 首页仪表盘 | / | 总览数据 |
| 3 | 设备列表 | /devices | 设备管理 |
| 4 | 设备详情 | /devices/:id | 单设备详情 |
| 5 | 实时监控 | /monitor | 实时数据 |
| 6 | 报警中心 | /alarms | 报警列表 |
| 7 | 报警详情 | /alarms/:id | 报警详情 |
| 8 | 工单列表 | /orders | 工单管理 |
| 9 | 工单详情 | /orders/:id | 工单详情 |
| 10 | 创建工单 | /orders/new | 新建工单 |
| 11 | 报表中心 | /reports | 数据报表 |
| 12 | 系统设置 | /settings | 系统配置 |
| 13 | 用户管理 | /settings/users | 用户管理 |

## 4. 色彩规范

| 用途 | 颜色 | 色值 |
|------|------|------|
| 主色 | 深蓝 | #1E3A5F |

| 成功 | 绿色 | #22C55E |

## 5. 组件库

| 组件 | 说明 |
|------|------|
| 设备卡片 | 显示设备基本信息、健康状态 |
| 健康进度条 | 显示健康分 0-100 |
| 报警徽章 | 颜色+数字显示报警数量 |
| 趋势图 | 折线图、面积图 |
| 饼图 | 分布统计 |
| 工单状态标签 | 待处理/处理中/已完成 |
| 时间线 | 设备历史事件 |

---

## 6. 响应式设计

| 断点 | 宽度 | 布局 |
|------|------|------|
| 移动端 | < 768px | 单列，卡片式 |
| 平板 | 768-1024px | 双列 |
| 桌面 | > 1024px | 多列，表格+侧边栏 |
