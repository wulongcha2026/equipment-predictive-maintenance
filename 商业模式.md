# 商业模式细化

## 1. 核心价值主张

| 价值点 | 具体说明 |
|--------|----------|
| **减少停机** | 提前 1-7 天预警，减少非计划停机 30%+ |
| **节省成本** | 预测性维护 vs 被动维修，节省维修费用 20%+ |
| **延长寿命** | 科学维保，设备寿命延长 10-20% |
| **降本增效** | 减少过度维保，优化备件库存 |

---

## 2. 定价策略

### 2.1 产品版本

| 版本 | 功能 | 价格 | 目标客户 |
|------|------|------|----------|
| **基础版** | 设备接入 + 实时监测 + 基础报警 | 300元/台/年 | 小型工厂 |
| **专业版** | 基础版 + AI 预测 + 工单管理 | 800元/台/年 | 中型工厂 |
| **企业版** | 专业版 + 定制报表 + API 集成 | 1500元/台/年 | 大型工厂 |

### 2.2 硬件销售

| 组件 | 价格 |
|------|------|
| 传感器套装 (振动+温度+电流) | 500-800元/套 |
| 边缘网关 (树莓派预装) | 600元/台 |
| 安装调试费 | 200-500元/台 |

### 2.3 收入估算 (初期)

| 场景 | 数量 | 收入 |
|------|------|------|
| 试点 1 家工厂 | 10 台设备 | 硬件: 1.3万 + 软件: 8000元/年 |
| 复制 5 家工厂 | 50 台设备 | 硬件: 6.5万 + 软件: 4万/年 |
| 扩展到 20 家 | 200 台设备 | 硬件: 26万 + 软件: 16万/年 |

## 3. 获客渠道

| 渠道 | 方式 | 成本 | 转化周期 |
|------|------|------|----------|
| **行业展会** | 参加工业展会 | 高 (5-10万) | 短 |
| **地推** | 扫工业园区 | 中 | 中 |
| **转介绍** | 现有客户推荐 | 低 | 短 |
| **内容营销** | 公众号、短视频 | 低 | 长 |
| **SaaS 平台** | 官网自助购买 | 低 | 长 |

**建议**: 初期以 **地推 + 转介绍** 为主

---

## 4. 销售流程

```
1. 线索获取
   ↓
2. 需求沟通 (了解工厂设备情况)
   ↓
3. 现场调研 (免费/低价)
   ↓
4. 方案报价
   ↓
5. 试点签约 (1-2 台设备试用)
   ↓
6. 部署实施 (1-2 周)
   ↓
7. 验收付费
   ↓
8. 续费/增购
```

## 5. 成本结构

| 成本项 | 金额 (月) | 说明 |
|--------|----------|------|
| 服务器 | 500-1000元 | 阿里云/腾讯云 |

| 销售费用 | 不定 | 提成、展会 |
| 办公 | 0-2000元 | 远程/共享办公 |

**初期月成本**: 约 1-2 万元

## 7. 竞争壁垒

| 壁垒 | 说明 |
|------|------|
| **数据积累** | 设备越多，模型越准 |
| **行业经验** | 懂工业、懂设备 |
| **先发优势** | 早期建立客户信任 |
| **服务能力** | 响应快、服务好 |

---

## 8. 风险与对策

| 风险 | 等级 | 对策 |
|------|------|------|
| 客户难找 | 🟡 中 | 聚焦细分行业，建立标杆 |
| 续费率低 | 🟡 中 | 提高服务质量，增加增值服务 |
| 技术难度 | 🟢 低 | 用成熟技术，不追求完美 |
| 资金压力 | 🟠 高 | 控制成本，小步快跑 |
